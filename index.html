Curso de Informática e Tecnologias Multimédia no Colégio de Gaia

Frequentei curso com Plano Próprio de Informática e Tecnologias Multimédia que agrega as vertentes informática, multimédia, gestão de redes e bases de dados.
O plano curricular é estruturado de forma a permitir aos alunos escolherem os seus caminhos acadêmicos e profissionais para várias áreas, como programação, tecnologia web e dispositivos móveis, bases de dados, gestão de infraestrutura informática (redes e segurança) e jogos digitais. Após a sua conclusão, é atribuído ao aluno um diploma do QNQ de nível 4, podendo prosseguir estudos para o ensino superior ou entrar para o mercado de trabalho.

Competências digitais:


Programação:

  • Desenvolvimento de aplicações em Visual Basic
  • Programação Orientada a Objetos em C#
  • Criação de websites e aplicações web utilizando HTML, CSS, JS e PHP
  • Desenvolvimento de jogos em Unity

Bases de dados:

  • Criação de bases de dados relacionais
  • Conhecimentos de Sistemas Gestores de bases de dados: phpMyadmin, MySQL, Access
  • SQL

Redes de Computadores e hardware:

  • Instalação, configuração e manutenção de computadores, periféricos, redes informáticas e sistemas operativos.
  • Instalação e configuração de servidores.
  • Instalação, configuração e administração de software de gestão de redes informáticas.

Ferramentas:

  • Microsoft Office: Word, Powerpoint, Excel, Publisher
  • Edição e imagem: photoshop, canva
  • Edição de vídeo: Davinci Resolve, Capcut
  • Ferramentas de produtividade: Trello, Jira, MS Project
