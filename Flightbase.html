FlightBase - Projeto

Descrição do Projeto

Este projeto foi desenvolvido no 11º, na disciplina de Técnicas de Programação em C#. O FlightBase é uma aplicação que guarda dados de voos realizados pelo utilizador. Este pode também editar sempre que quiser os dados de voos realizados. O projeto conta ainda com uma base de dados em Access para guardar os dados

Com este projeto aprimorei o meu conhecimento em Programação Orientada ao Objeto em C#. Além disso melhorei também o meu conhecimento acerca da tecnologia ADO.NET e bases de dados relacionais.
